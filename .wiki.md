# Project Summary
ScoutTalent is an AI-powered football talent marketplace that connects African youth players with global scouts. The platform facilitates player scouting, profile management, and enhances visibility for aspiring athletes through a microservices architecture, which now includes a Media Service for video uploads and management.

# Project Module Description
The project consists of the following functional modules:
- **Auth Service**: Handles user registration, authentication, and role management.
- **Profile Service**: Manages player and scout profiles, including details, trust levels, and profile completion scoring.
- **Media Service**: Manages video uploads and metadata, providing functionalities for users to showcase their skills.

# File Description Inventory
- **Makefile**: Contains commands for building, testing, and managing services.
- **docker-compose.yml**: Defines the services for the application and their configurations.
- **init-databases.sql**: Script to create databases for each service.
- **QUICKSTART.md**: Quick start guide for setting up and running the project.
- **README.md**: Overview of the project, architecture, and services.
- **scripts/test-api.sh**: Script to automate API testing for Auth and Profile services.
- **scripts/test-media-api.sh**: Script to automate API testing for Media Service.
- **LOCAL_TESTING_GUIDE.md**: Guide for setting up and testing the platform locally.

# Technology Stack
- **Backend**: Go 1.22+ with Gin framework
- **Databases**: PostgreSQL 18.1
- **Cache**: Redis 8.4
- **Message Queue**: NATS JetStream
- **Containerization**: Docker
- **Orchestration**: Kubernetes (AKS)

# Usage
### Prerequisites
- Go 1.22+
- Docker & Docker Compose
- Node.js 20+
- golang-migrate

### Quick Start
```bash
# 1. Start all services
make dev

# 2. Run database migrations
make migrate-up

# 3. Test the services
./scripts/test-api.sh          # Auth & Profile
./scripts/test-media-api.sh    # Media Service
```

### Manual Testing
Refer to **LOCAL_TESTING_GUIDE.md** for detailed manual testing instructions.
